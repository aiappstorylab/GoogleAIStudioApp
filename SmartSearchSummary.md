# 📋 앱 이름: 스마트 검색 서머리 (Smart Search Summary)

## 1. 앱 소개 및 개요
이 앱은 사용자가 궁금한 주제나 키워드를 입력하면, 구글 실시간 검색을 통해 방대한 정보를 분석하고 핵심만 구조화하여 요약해 주는 AI 검색 보조 앱입니다.

## 2. 사용자 스토리보드
- **입력:** 사용자는 검색하고 싶은 복잡한 주제(예: 최신 IT 트렌드, 건강 정보 등)를 입력합니다.
- **분석:** AI가 구글 검색을 통해 신뢰도 높은 상위 문서들을 실시간으로 읽습니다.
- **결과:** 사용자는 수많은 링크를 클릭할 필요 없이, 단 10초 만에 정리된 리포트를 확인합니다.
- **확장:** 요약문 하단의 출처 링크를 통해 필요한 경우 원문으로 바로 이동합니다.

## 3. 기능적 요구사항 (Input & Data)
- **사용자 입력 (Input):** 검색 키워드 또는 질문 문장.
- **AI 처리:**
  - **Google Search Tool:** 구글 검색 엔진과 연동하여 최신 데이터 수집.
  - **Summarization:** 수집된 텍스트 중 중복을 제거하고 핵심 가치(Value) 위주로 재구성.
- **출력 (Output):** - 1단계: 한 줄 핵심 요약.
  - 2단계: 3가지 핵심 포인트 (불렛포인트).
  - 3단계: 팩트 체크 및 장단점 비교 대조군 생성.
  - 4단계: 정보의 신뢰도를 위한 원문 출처 리스트 제공.

## 4. 제약사항 및 규칙
- **검색 기반 답변:** AI가 미리 학습한 지식이 아닌, 반드시 실시간 검색 결과를 바탕으로 답변합니다.
- **환각(Hallucination) 방지:** 검색 결과에 없는 내용을 지어내거나 추측하지 않습니다.
- **가독성 최적화:** 전문 용어는 쉽게 풀어서 설명하며, 한국어 응답을 기본으로 합니다.
