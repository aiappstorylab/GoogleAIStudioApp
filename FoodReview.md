Project: AI 맛집 리뷰 크리에이터 (AI Food Reviewer)

"사진 한 장으로 끝내는 고퀄리티 SNS 리뷰 생성 에이전트"

1. 앱 소개 (Introduction)

이 프로젝트는 사용자가 촬영한 음식 및 장소 이미지를 Gemini 2.5 Vision으로 분석하고, 실시간 구글 검색 데이터를 결합하여 감성적인 인스타그램 피드나 상세한 블로그 포스팅, 배달앱 리뷰를 자동으로 생성해 주는 AI 도구입니다.

2. 사용자 스토리보드 (Storyboard)

사용자는 맛집에서 찍은 사진들을 업로드합니다.

AI는 사진 속 메뉴, 재료, 매장 인테리어를 정교하게 분석합니다.

AI는 구글 검색을 통해 해당 식당의 최신 정보(가격, 위치 등)를 보완합니다.

사용자는 완성된 리뷰를 복사하여 SNS에 바로 게시합니다.

3. 기능적 요구사항 (Requirements)

3.1 입력 (Input)

음식 사진, 매장 내부 사진, 메뉴판 사진 (멀티모달 대응).

음식점 상호명, 음식에 대한 느낌(텍스트로 입력받음).

이미지의 경우 용량을 200kb 이하로 지정, 이미지 용량이 크면 이미지 사이즈를 조절하여 용량을 줄이는 작업 수행.

원하는 리뷰 톤 (인스타 감성형 / 블로그 정보형 / 배달앱 리뷰형 / 솔직 담백형)

3.2 처리 (Processing)

Vision Analysis: 이미지 내 객체 식별 및 텍스트 OCR

Context Synthesis: 사진의 무드와 검색 데이터를 결합한 문장 생성

Search Integration: Google Search Tool을 통한 팩트 체크

3.3 출력 (Output)

플랫폼별(SNS, 블로그, 배달앱 리뷰) 최적화된 본문 텍스트

관련 해시태그 추천 리스트

정보 출처(링크) 제공

4. 제약사항 (Constraints)

사진에 없는 허위 정보를 생성하지 않습니다.

특정 장소의 영업 상태(폐업 등)를 반드시 검색 결과로 대조합니다.

한국어 및 영어 출력을 기본 지원합니다.
